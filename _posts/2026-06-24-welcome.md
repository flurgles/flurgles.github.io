This is the first post on the dock6 development update log. I'll use this site
to document progress, share highlights, and keep a running record of what's
changing in my development branch of DOCK 6.

<!--more-->

## What's been happening

The dev branch has been active for a while. Here's a quick summary of the major
areas of work so far:

### GPU Acceleration (Metal)

Adding GPU support using Apple's Metal framework — initially for scoring
kernels, now extended to a full GPU simplex minimizer for flexible docking.
Current work includes speculative batch simplex evaluation and performance
tuning.

### RMSD & Clustering

New WL graph symmetry detection with Hungarian algorithm min-cost matching for
RMSD calculations. The `cluster_rmsd_type` option has been split into
`pruning_cluster_rmsd_type` and `final_pose_cluster_rmsd_type` for finer
control.

### Build System

Added `clang` and `clang.gpu` configuration templates for Apple Silicon,
cleaned up stale build artifacts, and untracked generated files from version
control.
